# IPDB

## An IP address management database

### Status

Work in progress:
* Layout improvements.
* Extra table/field configuration dialogs.

### Requirements

* A web server.
* PHP >= 5.4.

### Installation

* Copy the file `config.dist.php` to `config.php`.
* Edit `config.php`.
* Create database.
* Point your browser to `<...>/ipdb/`.
